GII THIU V HTML
S RA I CA HTML
HTML (HyperText Markup Language) l mt ngn ng
nh du c s dng to cu trc v hin th ni dung
trn cc trang web. N cho php cc nh pht trin web xc
nh v nh dng cc phn t trong trang web, nh tiu ,
on vn bn, hnh nh, bng, lin kt v cc yu t khc.
HTML s dng cc th (tags) bao bc v xc nh cc
phn t trong trang web.
HTML ra i vo nm 1990 bi Tim Berners-Lee, mt
nh khoa hc my tnh ngi Anh. Khi , Tim Berners-Lee
lm vic ti CERN (T chc Nghin cu Ht nhn Chu u)
Thy S v ng mun to ra mt h thng cho php cc nh
nghin cu truy cp v chia s thng tin nhanh chng thng
qua mng Internet.
HTML ban u ch n gin l mt phin bn nh du
vn bn, cho php lin kt gia cc ti liu bng cch s
dng siu lin kt (hyperlink). Ngn ng ny c thit k
to ra cc trang web tnh, ngha l cc trang khng thay i
ni dung da trn hnh ng ca ngi dng.
Tuy nhin, vi s pht trin ca Internet v yu cu ngy
cng cao v tnh tng tc v a phng tin trn web, HTML
tri qua nhiu phin bn v tin ha theo thi gian. Mi
phin bn mi a ra cc ci tin v tnh nng mi p
ng nhu cu pht trin trang web phc tp hn.
8
Phin bn HTML4, c cng b vo nm 1997, gii
thiu cc yu t mi nh khung (frames), biu mu (forms) v
nh dng trang (layout). N m ra kh nng to ra cc
trang web c cu trc phc tp hn v tng tc vi ngi
dng.
HTML5, phin bn mi nht v c cng b vo nm
2014, a ra mt bc tin ln trong vic cung cp tnh
nng v kh nng mi cho pht trin web. HTML5 h tr a
phng tin, ha v tng tc ngi dng mt cch mnh
m hn. N cung cp cc yu t mi nh video, m thanh,
ha vector (Canvas v SVG), lu tr d liu cc b (Local
Storage), kt ni mng (WebSockets) v nhiu tnh nng khc.
HTML5 gip nng cao tri nghim ngi dng trn
web, cho php to ra cc ng dng web phc tp v p ng
trn nhiu thit b khc nhau.
HTML tr thnh mt chun cng cng v c s dng
rng ri trn ton th gii. N m ra cnh ca cho s pht
trin ca web v to nn s kt ni ton cu thng qua cc
trang web v ng dng trc tuyn. HTML khng ch l mt
phn quan trng ca vic thit k v xy dng trang web, m
cn l nn tng cho victo ra cc cng ngh v ngn ng
khc nh CSS (Cascading Style Sheets) v JavaScript, ci
thin giao din v tng tc trn trang web.
CSS c s dng nh dng v trnh by cc phn t
HTML, cho php iu chnh mu sc, kch thc, v tr v hiu
ng ca cc phn t trn trang web. JavaScript, mt ngn
ng lp trnh pha my khch, cho php to ra cc tng tc
ng v thay i ni dung ca trang web da trn hnh ng
ca ngi dng.
HTML tip tc c pht trin v ci tin, vi cc phin
bn mi v cc tiu chun mi nhm p ng cc xu hng
9
v yu cu ca ngnh cng nghip web. Cng ng pht
trin web v cc t chc chun ha nh W3C (World Wide
Web Consortium) tip tc lm vic a ra cc tiu chun
v hng dn cho vic s dng HTML mt cch hiu qu v
tin tin.
HTML5 t c s chp nhn rng ri v c h tr
trn hu ht cc trnh duyt web hin i. N thc y s
pht trin ca cc ng dng web tin tin, tri nghim ngi
dng tt hn v to ra mi trng web a phng tin phong
ph.
Tm li, s ra i ca HTML vo nm 1990 m ra cnh
ca cho vic pht trin web v to nn s kt ni ton cu
thng qua cc trang web v ng dng trc tuyn. HTML
tin ha v c ci tin theo thi gian, vi phin bn HTML5
l phin bn mi nht v mnh m nht cho vic pht trin
web hin i.
CC PHIN BN HTML
1. HTML 2.0: Phin bn HTML 2.0 c cng b vo nm
1995. y l phin bn u tin c cng nhn l mt tiu

2. HTML 3.2: Phin bn HTML 3.2 c cng b vo nm
1997 bi W3C. N b sung nhiu yu t mi nh bng, mu
(form), khung (frames) v nh dng trang (layout). HTML 3.2
cung cp cc cng c to ra cc trang web phc tp hn

3. HTML 4.01: Phin bn HTML 4.01 c cng b vo
nm 1999. N l mt phin bn nng cp t HTML 3.2 v

chnh giao din ca trang web. HTML 4.01 cng h tr cc
10
yu t ng nh mu (form) v kch bn (script) to ra cc
trang web tng tc.
4. XHTML 1.0: XHTML (Extensible HTML) l mt phin bn
ca HTML c to ra bi W3C vo nm 2000. XHTML kt
hp c php ca HTML vi c php XML (eXtensible Markup
Language). XHTML 1.0 nh ngha mt cch cht ch hn c
php v quy tc ca HTML m bo tnh nht qun v
tng thch vi cc cng ngh web khc.
5. HTML5: HTML5 l phin bn mi nht v mnh m nht
ca HTML. N c cng b vo nm 2014 bi W3C. HTML5
a ra nhiu tnh nng v ci tin mi, bao gm h tr a
phng tin (video, m thanh), ha vector (Canvas v

(WebSockets) v nhiu tnh nng khc. HTML5 cung cp s
linh hot v kh nng tng tc cao hn, cho php xy dng
cc ng dng web phc tp v tri nghim ngi dng tt
hn.
Cc phin bn HTML tip tc c pht trin v ci tin
theo thi gian, vi cc phin bn nh v cp nht nhm p
ng cc xu hng v yu cu ca ngnh cng nghip web.

CA MT TI LIU HTML
Mt ti liu HTML (Hypertext Markup Language) c bn
bao gm cc thnh phn chnh sau:
1. Doctype: y l ch dn u tin trong ti liu HTML v
n thng bo vi trnh duyt v phin bn HTML m ti liu
s dng. V d, <!DOCTYPE html> cho bit y l mt ti
liu HTML5.
11
2. Th <html>: Th <html> nh du u v cui ca ti
liu HTML v cha ton b ni dung ca trang web.
3. Th <head>: Th <head> cha thng tin v ti liu
HTML nh tiu , nh dng trang, cc siu d liu (meta
data), kt ni ti cc tp CSS v JavaScript, v cc thng tin
khc khng c hin th trc tip trn trang web.
4. Th <title>: Th <title> t tiu ca trang web
v ni dung trong th ny s c hin th trong tiu ca
s trnh duyt hoc trong kt qu tm kim.
5. Th <body>: Th <body> cha ton b ni dung hin
th trc tip trn trang web, bao gm vn bn, hnh nh, lin
kt, bng, mu (form), v cc yu t khc.
Di y l mt v d v cu trc c bn ca mt ti liu
HTML:
<!lDOCTYPE html>
<html>
<head>
<title>Tiu trang web</title>
</head>
<body>
<h1>y l mt tiu </h1>
<p>y l mt on vn bn.</p>
</body>
</html>
Trong v d trn, ti liu HTML bt u bng <!DOCTYPE
html> ch nh rng y l mt ti liu HTML5. Tiu
ca trang web c t trong th <title>. Phn ni dung
trc tip ca trang web c t trong th <body>, bao gm
mt tiu (<h1>), mt on vn bn (<p>), mt hnh nh
(<img>), v mt lin kt (<a>).
